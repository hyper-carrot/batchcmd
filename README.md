go_command
==========

A Helper for executing system command.

# Usage

1. Download or clone this repo.

2. Run & fun:
```bash
cd <repo_path>
go run icommand.go -c=<command> 
```
I often use it with `-c="git pull"` for pulling the content in subdir from git repo.
